##  معرفی پروژه

این یک اپلیکیشن موبایل **مدیریت کارها (Todo List)** است که با استفاده از **.NET MAUI Blazor Hybrid** ساخته شده است. برنامه به صورت کاملاً آفلاین کار می‌کند و اطلاعات کاربران را در خود گوشی ذخیره می‌کند.

---

##  ابزارها و تکنولوژی‌های استفاده شده

| ابزار | نسخه | کاربرد |
| :--- | :--- | :--- |
| .NET MAUI | 6.0 | فریمورک اصلی برای ساخت اپ موبایل |
| Blazor Hybrid | 6.0 | ساخت رابط کاربری با HTML/CSS |
| C# | 10.0 | منطق برنامه و سرویس‌ها |
| Bootstrap | 5.3 | استایل و طراحی رابط کاربری |
| Preferences API | - | ذخیره‌سازی محلی داده‌ها |
| Git | - | کنترل نسخه |

##  ساختار پروژه
TodoApp/
├── Data/ # لایه داده
│ ├── TodoItem.cs # مدل داده (ساختار هر کار)
│ └── TodoStorage.cs # عملیات ذخیره‌سازی
├── Services/ # لایه سرویس
│ ├── ExportService.cs # خروجی/ورودی JSON
│ └── NotificationService.cs # اعلان‌ها
├── Pages/ # صفحات برنامه
│ ├── Index.razor # صفحه شروع (هدایت)
│ ├── Todos.razor # صفحه اصلی لیست کارها
│ ├── AddEditTodo.razor # افزودن/ویرایش کار
│ ├── Completed.razor # کارهای انجام شده
│ └── Statistics.razor # آمار و نمودار
├── Shared/ # اجزای مشترک
│ ├── MainLayout.razor # قالب اصلی
│ └── NavMenu.razor # منوی کناری
├── wwwroot/ # فایل‌های وب
│ └── css/app.css # استایل سفارشی
├── MauiProgram.cs # تنظیمات سرویس‌ها
├── MainPage.xaml # میزبان BlazorWebView
└── Main.razor # مسیریابی

##  ویژگی‌های کلی برنامه

| شماره | قابلیت | توضیح |
| :--- | :--- | :--- |
| 1 | افزودن کار جدید | عنوان، توضیحات، تاریخ، ساعت، اولویت، دسته‌بندی |
| 2 | ویرایش کار | تغییر اطلاعات کارهای موجود |
| 3 | حذف کار | حذف کارها با تأیید کاربر |
| 4 | انجام شده | علامت زدن کارهای انجام شده |
| 5 | پین کردن | نمایش کارهای مهم در بالای لیست |
| 6 | جستجو | جستجو بر اساس عنوان |
| 7 | فیلتر تاریخ | امروز، این هفته، این ماه |
| 8 | فیلتر دسته‌بندی | کاری، شخصی، خرید، سلامتی |
| 9 | اولویت‌بندی | کم (سبز)، متوسط (نارنجی)، بالا (قرمز) |
| 10 | آمار | نمایش درصد پیشرفت و تعداد کارها |
| 11 | خروجی JSON | پشتیبان‌گیری از کارها |
| 12 | ورودی JSON | بازیابی از فایل پشتیبان |
| 13 | اعلان | یادآوری کارهای امروز |




##  نحوه اجرای پروژه

### پیش‌نیازها:
- Visual Studio 2022 (نسخه 17.4 یا بالاتر)
- .NET 6.0 SDK
- بار کاری (Workload) ".NET MAUI"

### مراحل اجرا:
1. پروژه را با Visual Studio باز کنید
2. از نوار ابزار، دستگاه مورد نظر را انتخاب کنید (Windows Machine یا Android Emulator)
3. کلید **F5** را بزنید
4. منتظر بیلد و اجرا شدن برنامه باشید
